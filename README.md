# 🎓 Examination Seat Allotment System

## 📌 Project Overview

The **Examination Seat Allotment System** is a smart web-based application developed to automate the process of examination seating arrangements in colleges and educational institutions.

The system helps administrators manage student classes, examination rooms, and automatically generate organized seating plans. It reduces manual work, saves time, minimizes errors, and provides an efficient solution for exam management.

---

## 🚀 Features

✅ Student database management  
✅ Room database management  
✅ Select multiple classes for examination  
✅ Automatic seat allotment  
✅ Room-wise seating arrangement  
✅ Student roll number allocation  
✅ Capacity calculation  
✅ Unallocated student detection  
✅ Editable seating plan  
✅ Print seating arrangement  

---
